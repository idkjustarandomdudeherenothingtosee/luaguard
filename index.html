Multi-Layer Code Encryption
How it works: Your code goes through 5 layers of encryption:
1. Shamir's Secret Sharing (Key Fragment)
2. XChaCha20 encryption
3. RSA encryption for XChaCha20 key
4. Twofish encryption
5. AES-256 final layer
Then it's securely stored on Pastefy
